# 👁 The Eye
### AI-Powered Anti-Corruption Monitoring Platform

> *"Nothing Hidden. Everything Accountable."*

**Live Demo:** [theeye.vercel.app](https://theeye.vercel.app)

---

## What Is The Eye?

The Eye is a tamper-proof, AI-powered accountability platform that runs silently on organisational systems — recording every file access, payment, permission change, database query, login, and financial transaction into an immutable, cryptographically-chained audit trail that no administrator can delete or alter.

Built specifically for African governments, municipalities, state-owned enterprises, and corporations where corruption costs $88 billion annually and zero purpose-built monitoring infrastructure exists.

---

## Features

- **Universal Activity Monitoring** — every login, file change, payment, database query recorded
- **Immutable Audit Trail** — cryptographically hash-chained, tamper-proof logs
- **AI Intelligence Engine** — ask plain-English questions, get forensic answers instantly
- **Real-Time Intrusion Detection** — brute force, credential stuffing, Tor/VPN detection
- **Live Threat Map** — visualise attack origins in real time
- **MFA + Account Lockout** — 3 strikes and the attacker is blocked and reported
- **Multi-Person Deletion Approval** — no single person can destroy evidence
- **Forensic Investigation Module** — court-ready evidence export, NPA submission
- **Risk Scoring** — every user gets a live AI-calculated risk score (0–100)
- **Whistleblower Portal** — anonymous tip submission with AI cross-referencing

## Getting Started

```bash
# Clone the repo
git clone https://github.com/yourusername/the-eye.git
cd the-eye

# Install dependencies
npm install

# Run locally
npm run dev

# Build for production
npm run build
```
